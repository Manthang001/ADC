# Pipeline & Time-Interleaved ADC Design

**Duration:** May 2026 – Present  
**Status:** Ongoing (Research & Learning Phase)

## Overview

This project is being carried out under the guidance of **Prof. Neeraj Mishra** and **Prof. Anu Gupta** (Department of Electrical & Electronics Engineering).

The objective is to study, design, and eventually implement a high-speed Pipeline Analog-to-Digital Converter (ADC) integrated with Time-Interleaved (TI) ADC architectures. The project aims to explore techniques for improving sampling rates while maintaining high accuracy through calibration and efficient clock generation.

## Project Objectives

- Study the architecture and operation of Pipeline ADCs.
- Understand calibration techniques used to improve ADC accuracy.
- Explore 4-, 8-, and 16-channel Time-Interleaved ADC architectures.
- Design multiphase clock generation circuits for TI ADCs.
- Integrate the clock generator with the ADC architecture for high-speed data conversion.
- Evaluate design trade-offs such as speed, power, resolution, and mismatch errors.

## Current Progress

The project is currently in its initial phase, focusing on:

- Literature review of Pipeline ADC architectures
- Understanding Time-Interleaved ADC design principles
- Studying calibration methods
- Reviewing multiphase clock generation techniques
- Defining the overall system architecture

Implementation, simulation, and hardware design will be carried out in subsequent phases.

## Planned Workflow

1. Study Pipeline ADC architecture
2. Design and analyze individual pipeline stages
3. Develop calibration techniques
4. Explore 4-, 8-, and 16-channel TI ADC architectures
5. Design multiphase clock generators
6. Integrate the clock generator with the ADC
7. Simulate and evaluate system performance
